## üìñ About The Project

This repository hosts the source code for a **Basic Arithmetic Calculator**. The project was developed as a key task during my internship to demonstrate proficiency in Java logic and user input handling.

It features a user-friendly command-line interface developed using **Java Standard Library (java.util)**, ensuring precision and reliability without the need for external dependencies.

### üåü Key Features
* **Menu-Driven Interface:** Prompts the user to enter numbers and select operations via a clean console menu.
* **Arithmetic Logic:** Implements **Switch-Case** statements to handle Addition, Subtraction, Multiplication, and Division efficiently.
* **Error Handling:** Includes specific logic to detect and prevent **Division by Zero** errors, ensuring the program doesn't crash.
* **Input Validation:** Utilizes `Scanner` class to handle double-precision floating-point numbers for accurate calculations.

## üöÄ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites
* **Java Development Kit (JDK)** installed on your system.
* An IDE (VS Code, IntelliJ, or Eclipse) or a simple text editor.

### Installation & Execution

1.  **Clone the repository**
    ```sh
    git clone https://github.com/Sumitkolgire23/CODTECH-Task1
    ```
2.  **Navigate to the project directory**
    ```sh
    cd Basic-Calculator
    ```
3.  **Compile the Java file**
    ```sh
    javac BasicCalculator.java
    ```
4.  **Run the application**
    ```sh
    java BasicCalculator
    ```
